# CLAUDE.md (CloudMD)

Этот файл — краткая инструкция для **Claude Code** (и любых агентных CLI), чтобы изменения в репо оставались предсказуемыми.

## Цель

Поддерживать **современный однопользовательский Todo-лист** для демо: статическое приложение без зависимостей.

## Жёсткие ограничения

- **Без зависимостей и сборки**: не добавлять `package.json`, npm/yarn/pnpm, bundlers.
- **Без GitHub Actions / workflows**.
- Приложение остаётся **статическим**: `index.html` + `style.css` + `app.js`.
- Данные храним в `localStorage` (ключ: `modern-todo-demo:v1`).

## Запуск

```bash
python3 -m http.server 8000
```

Открыть: `http://127.0.0.1:8000`

## Live demo (GitHub Pages)

Когда GitHub Pages включены, приложение доступно по адресу:

`https://OWNER.github.io/REPO/`

## Если меняешь поведение

- Обнови `README.md` и `DEMO.md` (если поменялись фичи/шаги демо).

## Если меняешь UI/UX

- Следуй `DESIGN_SYSTEM.md` (цвета/типографика/микро-анимации).
- Не используй emoji как иконки — только SVG.
